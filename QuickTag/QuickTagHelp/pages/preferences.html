The Preferences Window

When you select Preferences from the QuickTag menu, you'll be presented with the Preferences Window.

The 5 tabs are

  1. Genres: Specify the genre list.
  2. Categories: Specify the category list.
  3. Attributes: Specify the attribute list.
  4. Tagging: Configure QuickTag tagging.
  5. Options: Configure QuickTag options.

The "Apply" button will apply the changes. Settings are automatically saved in user preferences.
