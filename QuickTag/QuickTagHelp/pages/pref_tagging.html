The Tagging Preferences Tab

The taggin preferences tab allows you to edit the QuickTag tagging options:

Configurable options are:

  • Where to store tags: Comments, Category, Grouping (Default: Comments)
  • How to write the tags: Overwrite existing value, prepend (before), or append (after)
  • The delimiters to use for each different tag value (start and end), the enabled checkbox enables or disables each tag type.
  • Tag seperator: An additional delimiter to go in between each tag (Default: comma)
  • An additional tag line value to be added to each track's comments. This is useful for later determining later which tracks have already been tagged with QuickTag, or putting in your own name.
